# Resumo do que aprendi na Sprint

## Curso: Complete Introduction to Amazon QuickSight

Aprendi a usar o Amazon QuickSight para criar análises dinâmicas, relatórios e dashboards em minutos; preparar e relacionar dados para análise; construir visualizações avançadas; e aplicar cálculos de tabela (percentuais, totais acumulados, ranking e diferenças) para gerar insights acionáveis.

## Curso: Amazon QuickSight - Getting Started

Aprendi os conceitos técnicos e benefícios do Amazon QuickSight, incluindo sua arquitetura, estrutura de custos e casos de uso. Desenvolvi habilidades para projetar, criar e personalizar painéis interativos, integrando dados de múltiplas fontes para gerar insights de negócios.

## Material de apoio sobre Storytelling com dados

Aprendi os princípios do storytelling com dados, incluindo a importância de entender o público, definir o objetivo da comunicação, escolher visualizações apropriadas e criar narrativas envolventes. Desenvolvi habilidades para apresentar dados de forma clara e impactante, facilitando a tomada de decisões informadas.
