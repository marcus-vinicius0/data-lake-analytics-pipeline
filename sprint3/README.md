# Resumo do que aprendi na Sprint

## Modelagem de dados para Data Warehouse

Aprendi a modelar dados para Data Warehouse, trabalhando com dimensões, fatos e hierarquias, além de construir modelos multidimensionais Star Schema e Snowflake. Desenvolvi habilidades em operações OLAP, análise de indicadores, planejamento estratégico com foco em qualidade de dados e conceitos de Business Intelligence, compreendendo também a importância dos metadados.

## Amazon Q Developer for Programmers and DevOps AWS AI coding

Aprendi a utilizar o Amazon Q Developer para desenvolvimento de software, escrevendo e testando código rapidamente em Java, JavaScript, Python e Terraform. Desenvolvi habilidades em prompting eficiente, documentação de código e aproveitei recursos do Amazon Q e GitHub Copilot para produzir código de forma mais rápida e eficiente.
